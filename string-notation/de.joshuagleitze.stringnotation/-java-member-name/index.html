JavaMemberName

object JavaMemberName : BaseStringNotation (source)

A notation for java member names. This notation is like LowerCamelCase, but will drop any character that is not allowed in a Java identifier when printing.

Allowed characters are determined using Character.isJavaIdentifierStart and Character.isJavaIdentifierPart.
